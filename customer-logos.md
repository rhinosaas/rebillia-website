# Rebillia Customer Logos

This file tracks real customer logos for the "Trusted by Subscription Businesses" section.

## Instructions
1. Place logo files (SVG or PNG, transparent background) in a `/logos/` folder
2. Update the "Logo File" column with the filename
3. Once logo files are added, update `index.html` and `customers.html` to use `<img>` tags instead of text
